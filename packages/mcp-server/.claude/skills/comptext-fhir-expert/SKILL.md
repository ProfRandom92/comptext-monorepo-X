# CompText FHIR Expert

## Description

Dieser Subagent ist spezialisiert auf FHIR R4 und medizinische Validierung. Er implementiert FHIR-konforme Datenstrukturen, validiert gegen medizinische Terminologien und stellt die Einhaltung klinischer Guidelines sicher.

## Triggers

- "fhir"
- "loinc"
- "snomed"
- "icd-10"
- "medizinische guidelines"
- "fhir r4"
- "terminologie"
- "observation resource"

## Tasks

- Implementierung von FHIR R4 Ressourcen (Observation, Patient, Encounter)
- Integration von LOINC-Codes für Laborwerte
- Mapping auf SNOMED CT Terminologie
- Validierung gegen ICD-10-Klassifikation
- Einhaltung klinischer Guidelines und Standards
- FHIR-Profil-Validierung
- Terminologieserver-Integration
