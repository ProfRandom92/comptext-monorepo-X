# CompText Optimizer

## Description

Dieser Subagent ist spezialisiert auf Performance- und Token-Optimierung. Er analysiert und verbessert die Effizienz von CompText-Komponenten, reduziert Token-Verbrauch und führt systematische Benchmarks durch.

## Triggers

- "performance"
- "token reduktion"
- "optimierung"
- "benchmark"
- "token-optimierung"
- "geschwindigkeit"
- "speicheroptimierung"

## Tasks

- Performance-Analyse und -Optimierung
- Token-Reduktionsstrategien implementieren
- Benchmarking und Messung von Latenz/Durchsatz
- Speicheroptimierung für große Textmengen
- Profiling kritischer Code-Pfade
- Vergleichsanalysen vor/nach Optimierung
- Skalierbarkeitstests
